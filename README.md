## 📌 Advanced Library Management System (Kotlin - Console Based)

🚀 [Click here to run the application live in your browser!](https://replit.com/@rajeshgaikwad02/Library-Management-System)

A beginner-to-intermediate level, console-based Library Management System written in Kotlin. This project demonstrates object-oriented programming concepts with advanced features like book inventory tracking, member management, loan records, fine calculation, and overdue alerts. It simulates a real-world library system using a structured, menu-driven interface.

---

## ✨ Features

* **Book Management:** Add, view, search, and delete books with title, author, genre, and quantity.
* **Inventory Tracking:** Maintains total copies and available copies of each book.
* **Member Management:** Register, view, and delete library members with email details.
* **Borrowing System:** Members can borrow books with a limit of 3 books per member.
* **Return System:** Return books with automatic fine calculation for late returns.
* **Loan Records:** Tracks all borrowing and returning activities.
* **Fine System:** Calculates fine based on overdue days.
* **Overdue Alerts:** Alerts when books exceed the allowed loan period.
* **Usage Tracking:** Tracks how many times a book is borrowed.
* **Member Activity Tracking:** Tracks books held and total fines paid by members.
* **Day Simulation:** Advance days to simulate real-time borrowing and overdue conditions.
* **Menu-Driven Interface:** Clean and easy-to-use console interface.

## ▶️ How to Run Locally

1. **Save the Kotlin file** (e.g., `LibraryApp.kt`)

2. **Open Command Prompt / Terminal**

3. **Navigate to the folder**

```bash
cd path/to/your/folder
```

4. **Compile the Kotlin file**

```bash
kotlinc LibraryApp.kt
```

5. **Run the program**

```bash
kotlin LibraryAppKt
```
